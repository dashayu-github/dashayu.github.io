git 使用指南 windows 下载地址https://gitforwindows.org/index.html mac 下载地址:https://sourceforge.net/projects/git-osx-installer/ linux 输入命令 sudo apt-get install git （debian系统) 1.克隆GitHub上的仓库到本地方法 git clone +仓库链接 例如:git clone https://github.com/dashayu-github/Dmo.git 意本地的目录里没有链接里最后Dmo的名的目录，不然会报错 2.把本地文件同步到文件步骤1、git status 查看是否有更新的内容 2、git add +文件名 （添加要更新的文件或要同步的文件到暂存区） 3、 git commit -m "注释" 提交到本地仓库 4、git push 同步远程仓库
